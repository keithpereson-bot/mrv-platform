Setup Guide -- 5 minutes, no Cloud Console needed
1️⃣
Create a Google Sheet
Go to sheets.google.com and create a new blank spreadsheet. Name it MRV Platform Data.
2️⃣
Open Apps Script
In the sheet, go to Extensions -> Apps Script. Delete any existing code, then paste the entire MRV_AppScript.js file provided.
3️⃣
Deploy as Web App
Click Deploy -> New deployment. Type: Web app. Execute as: Me. Who has access: Anyone. Click Deploy and copy the URL.
4️⃣
Paste URL & Connect
Paste the Web App URL into the field above and click Connect & Test. Then push your data. Done!
!️ When deploying, set Execute as: Me and Who has access: Anyone. This lets the app read/write your sheet without requiring users to sign in. The URL acts as a secret key -- keep it private.
